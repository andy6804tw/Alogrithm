# 01-Knapsack
0-1背包的解可以編碼為一串0-1字串（0：不取，1：取），首先隨機產生 M 個 0-1 字串，然後評價這些 0-1 字串作為 0-1 背包問題的解的優劣；然後，隨機選擇一些字串通過交叉、突變等操作產生下一代的 M 個字串，而且較優的解被選中的概率要比較高。這樣經過G代的進化後就可能會產生出0-1背包問題的一個`近似最優解`。
